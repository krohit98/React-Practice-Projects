This project acts as a wrapper for some interesting React applications. 
Following 3 apps are contained in this:

1. Shopping Cart
   A shopping cart representation where the products are being fetched from an external API and users can:
	Add/Remove products from cart
	Increase/Decrease product quantity
	Hover over product images to get enlarged view 
	Get the total billing amount

2. Blinking Paragraph
   A paragraph where each word changes to a random color each second. 
   It utilizes the concepts of random color generation, setInterval and component cleanup method to build this interesting usecase.

3. Quiz App
   A basic quiz app with time limits for each question.
   It utilizes the concepts of useRef, setInterval and setTimeout to achieve this beautifully.
